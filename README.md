# archdash介绍

archdash是架构组用于区分前后端开发分离而设计的框架，主要用于解决以下几个问题：
1. 公司内部dwz框架开发缓慢，界面丑陋，问题较多等问题
2. 前后端开发分离，将js部分当做项目来做，为适配公司内部的组件化框架和长远发展做考虑
3. 选用文档丰富，社区资源丰富的框架进行扩展和开发方便公司后期招聘相关人才
4. 选用前后端均可以使用的开发框架开发后台，方便后续公司前后端人才可以互相整合

## 技术选型

1. 前端框架：Vue
2. 前端打包：webpack
3. 开发方式：node
4. 开发语言：支持es2015的js


## 基本使用

1. 安装环境

```bash
    make install
```

2. 开发测试

```bash
    make dev
```

3. 编译构建

```bash
    make build
```

4. 启动后端服务

```bash
    make start
```
